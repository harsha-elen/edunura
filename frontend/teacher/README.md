# LMS Teacher Portal

Teacher portal for managing courses and live classes.

## Features
- Course Content Management
- Live Class Scheduling (Zoom)
- Attendance Tracking
- Student Queries & Inbox

## Tech Stack
- React 18 + TypeScript
- Vite
- React Router v6
- Redux Toolkit
- React Query
- Material-UI

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

The teacher portal will run on http://localhost:3002
